# JavaScript

### 方法

#### scrolllntoView()

``` tex
该方法为js原生方法,调用它的素滚动到浏览器窗口的可见区域的顶部或底部,也可以百度scrolllntoView相关的各种插件. 注意: 根据其他元素的布局.元素可能无法完全滚动到顶部或底部.
```
